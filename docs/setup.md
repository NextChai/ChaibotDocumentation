## Setup
Setup custom channels for Chai updates! This used to be one command, but has since been moved to multiple commands for ease.

Setup your server for ChaiBot updates.

### **setupdatestaging**
Custom Update Staging Updates.

**Get notified when Epic Games Devs are working on the next Fortnite update.**

How to use:
```
chaisetupdatestaging channel[discord.TextChannel]
```

Use Example:
```
chaisetupdatestaging

chaisetupdatestaging #general
```

### **setaes**
Custom AES Updates.

**Get notified when the AES keys update.**

How to use:
